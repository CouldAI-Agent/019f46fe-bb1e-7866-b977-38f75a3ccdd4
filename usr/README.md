# Fullstack Developer Portfolio

A modern, responsive portfolio application for a fullstack developer. Built to showcase skills, projects, and contact information across all device sizes.

## Features
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Hero Section**: A welcoming introduction with a clear call to action.
- **Skills Area**: A visual representation of technical proficiencies across frontend, backend, and tools.
- **Projects Showcase**: Interactive cards detailing past work, complete with tech stack tags and links.
- **Contact Form**: A clean interface for potential clients or employers to get in touch.
- **Dark Theme**: A sleek, developer-friendly dark mode UI.

## Tech Stack
- Flutter (Cross-platform UI toolkit)
- Dart

## Setup
1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device or emulator.
